# ejs-render-browserify

[EJS](http://npmjs.org/package/ejs) precompiler for
[Browserify](http://browserify.org).

## Installation ##

``` bash
npm install ejs-render-browserify --save-dev
```

## Usage ##

``` javascript

b.transform('ejs-render-browserify', {/* data */}, { /* ejs options*/})

```
